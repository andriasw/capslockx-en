# Edge 增强模块（测试中）

| 模式     |  按键   | 说明                                  |
| -------- | :-----: | ------------------------------------- |
| 在Edge内 | Alt + w | 拿出笔（全屏模式暂时不支持）          |
| 在Edge内 | Alt + q | 换左边的笔/橡皮（全屏模式暂时不支持） |
| 在Edge内 | Alt + e | 换右边的笔/橡皮（全屏模式暂时不支持） |
| 在Edge内 | Alt + , | 上一章/节                             |
| 在Edge内 | Alt + . | 下一章/节                             |
| 在Edge内 | Alt + / | 显示目录                              |
| 在Edge内 | Alt + ; | 切换自适应页面大小模式                |
| 在Edge内 | Alt + ' | 切换双页布局模式                      |
