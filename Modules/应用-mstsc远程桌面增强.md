# MSTSC 远程桌面增强

| 模式             | 按键                    | 功能说明                                                         |
| ---------------- | :---------------------- | ---------------------------------------------------------------- |
| 任意窗口         | 双击左边 Shift+Ctrl+Alt | 后置当前窗口，用于远程桌面与虚拟机，使其可与本机桌面窗口同时显示 |
| 在远程桌面窗口中 | LAlt + RAlt             | 切换最小化当前远程桌面窗口                                       |
