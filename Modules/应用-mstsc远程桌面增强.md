# MSTSC 远程桌面增强

| 模式             | 按键                    | 功能说明                                       |
| ---------------- | :---------------------- | ---------------------------------------------- |
| 任意窗口         | 双击左边 Shift+Ctrl+Alt | 最小化当前窗口（可穿透远程桌面与虚拟机）       |
| 在远程桌面窗口中 | LAlt + RAlt             | 最小化当前远程桌面窗口：或回到远程桌面操作界面 |
